## Notes for Developers

**This is the single source of truth for all student data.**

When updating student information:
1. Update this file FIRST
2. Then sync to:
   - `students.html` STUDENTS array
   - `foundation.html` FOUNDATION_STUDENTS array
   - `index.html` FOUNDATION_STUDENTS array (for attendance)
   - `student-profile.html` STUDENTS array

**Saturday Class Timeslot Rules:**
- Students can only be in ONE timeslot (AM or PM) or None
- Foundation students should mostly be in Saturday Class
- Some advanced team members may not attend Saturday Class

**Team Assignment Rules:**
- Foundation = `team: null` (unassigned)
- EV Only = `team: null` but `evTeam: true`
- Competition teams = `team: '839Z'/'839Y'/'839X'`
- Students can be on VRC team + EV team simultaneously
